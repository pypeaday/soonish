# Notifiq

This monorepo contains several directories with different purposes

- `backend` - The main backend for notifiq
    - includes a website built with HTMX and Alpine.js modleing the backend features
    - requires temporal server

- `frontend-ABC` will be a pattern for the various front-ends I want to build on top of notifiq

## Development

Currently development is very bespoke and bare-metal. Docker and containers are coming
